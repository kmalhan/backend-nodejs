## Node.js + MongoDB Sample API backend server

### Pre-requisite
- Node.js
- MongoDB

### Setup Instruction (OS X)
1. Install node.js, mongoDB, mongoose
```bash
brew install nodejs
brew install mongodb
brew install mongoose
```

### To Run
1. Start mongoDB server
2. Run main node.js file 
```bash
node index.js
```

### To Test
```bash
./apiTest.sh
```
